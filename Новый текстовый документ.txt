GET /users
200 - успешное получение всех пользователей
500 - ошибка сервера

GET /users/{id}
200 - успешное получение пользователя
400 - неверный запрос (например, если id не валидный)
404 - пользователь не найден
500 - ошибка сервера

POST /users
201 - успешное создание пользователя
400 - неверный запрос (например, если данные пользователя некорректны)
500 - ошибка сервера

PUT /users/{id}
200 - успешное изменение данных пользователя
400 - неверный запрос (например, если id не валидный или данные пользователя некорректны)
404 - пользователь не найден
500 - ошибка сервера

DELETE /users/{id}
204 - успешное удаление пользователя
400 - неверный запрос (например, если id не валидный)
404 - пользователь не найден
500 - ошибка сервера
